Get Nexa on Android

Requirements: Android 7.0 (API 24)+, Java 11, Android Studio, Firebase project.

Installation Guide

  1. Clone the repo: 
    git clone <repo-url>
  2. Open in Android Studio.
  3. Set up Firebase: register app, download google-services.json into /app.
  4. Build and run: 
    ./gradlew assembleDebug

FAQ

Is Nexa free?

Yes, completely free and open source.

What Android version?

Android 7.0 (API 24) or higher.

Does it work offline?

Yes, Firestore offline persistence is enabled by default.
