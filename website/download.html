Get Nexa on Android

Build and run your own instance of the ultimate personal daily super app.

Requirements

  • Android 7.0 (API 24) or higher
  • Firebase account (free tier is sufficient)
  • Android Studio (for building)
  • Java 11+

Installation Guide

1

Clone the repository

Download the source code to your local machine.

git clone https://github.com/monikdiwani/Nexa.git
2

Open in Android Studio

Open Android Studio, select "Open an existing project", and choose the cloned Nexa folder. Let Gradle sync completely.

3

Firebase Setup

Nexa relies on Firebase for real-time sync, auth, and storage.

Important: Register the package name com.fairshare.app in your Firebase Console project settings.

Download your google-services.json file from Firebase and place it in the app/ directory of the project.

4

Build and Run

Connect your Android device or start an emulator, then run:

./gradlew assembleDebug

Frequently Asked Questions

Is Nexa free?
Yes, completely free and open source. There are no subscriptions, ads, or premium tiers.
What Android version is supported?
Nexa supports Android 7.0 (API level 24) and higher.
Does it work offline?
Yes, Firestore offline persistence is enabled. You can add tasks, notes, and expenses offline, and they will sync automatically when you reconnect.
Is my data private?
Because you set up your own Firebase project, you own 100% of your data. The app connects directly to your Firebase instance.
Can I contribute to the project?
Absolutely. Pull requests are welcome on GitHub for bug fixes, new features, or UI improvements.
