Complete Feature List

Every single feature built into Nexa, documented in detail.

🏠 Home Dashboard

Time-based greeting

Morning/Afternoon/Evening/Night greeting based on local time.

User name display

Shows Firebase Auth display name or email prefix.

Net balance card

Total Cash In minus Cash Out across all ledgers.

Cash In / Out breakdown

Visual breakdown inside the balance card.

Smart insight message

Dynamic AI-like tip based on spending, tasks, and reminders.

Quick Actions

Buttons to instantly go to Money Tab, Add Task, Set Reminder, or View Reports.

Tasks preview card

Shows pending count + next upcoming task title.

Reminders preview

Shows upcoming count + next reminder + trigger time.

Ledgers preview

Displays total ledger count and quick tap to Money tab.

Global search shortcut

Search icon directly from home screen header.

Real-time listeners

Firestore data updates instantly without pull-to-refresh.

Exit protection

Double back-press required to exit the app.

💰 Money & Finance

3-tab view

Filter ledgers by Personal, Shared, or All.

Pending settlements banner

Alerts appear when you owe someone in a shared ledger.

Create Ledger Book

Set name, currency, and auto-generates a 6-char invite code.

Join Ledger

Join existing ledgers using the 6-character invite code.

Filter transactions

Filter by medium (All/Cash/Bank) and time (Today/Week/Month).

Transaction categories

9 chips: Sales, Rent, Salary, Office, Personal, Food, Transport, Shopping, Bills.

Receipt scanner

ML Kit OCR extracts vendor name + total amount from photos.

SMS auto-add

Detects bank SMS and auto-logs transactions (if enabled).

Fair Share Splitting

Add shared expenses with EQUAL, PERCENTAGE, or EXACT split methods.

Debt simplification

Greedy algorithm provides minimum payment suggestions.

WhatsApp reminder

Send debt collection reminders directly to WhatsApp.

Export PDF report

Native Android PdfDocument API generation, no external libraries.

Recurring transactions

Auto-creation Daily, Weekly, Monthly, or Yearly.

Budget tracking

Set category limits, see progress with color-coded alerts (Green/Orange/Red).

Physical cash counter

Count denomination coins/notes to calculate total cash on hand.

📝 Smart Notes

Staggered grid layout

Samsung Notes-inspired colorful note card grid.

Note folders

Organize with Personal and Work folders in the left drawer.

Rich text formatting

Bold, Italic, Underline, Strikethrough, Text Alignment.

Color highlighting

Change text color and background highlight colors.

Checklist mode

Convert body to checklist items, each with a clickable checkbox.

Undo / Redo

50-state history stack with 500ms debounce.

Image attachments

Attach images from gallery or snap a photo directly.

Page styles

Canvas-drawn backgrounds: Blank, Lined, Dotted, Grid.

Set reminder on note

Schedules an exact-minute alarm linked to the note.

Convert to other modules

Turn a note into a Task, Reminder, or Cashbook Entry with one tap.

Multi-select actions

Pin, archive, restore, trash, delete forever in batch mode.

Auto-save

Notes auto-save with timestamp tracking.

✅ Tasks

Priority sorting

Auto-sorts by High (Red) → Medium (Orange) → Low (Green).

Subtasks

Add multiple sub-items, each with its own completion checkbox.

Important flag

Star tasks to mark them as highly important.

Recurring tasks

Set patterns: None, Daily, Weekly, Monthly, Yearly.

One-tap complete

Instant strikethrough animation and color dimming.

Cross-module link

Link a task directly to a Note or Cashbook Entry.

🔔 Reminders & Alarms

Exact-minute firing

Uses AlarmManager.setExactAndAllowWhileIdle() for precision.

High priority bypass

High priority reminders bypass Do Not Disturb with sound/vibration.

Snooze

15-minute delay right from the notification action button.

Categories

Bill, Meeting, Task, Medicine, Shopping, Custom.

Daily Digests

Morning summary at 8 AM, Night review at 9:30 PM via WorkManager.

Boot Receiver

Reschedules ALL alarms automatically after a device restart.

📊 Reports & Analytics

Custom bar chart

Drawn on Android Canvas API. No external libraries used.

7 and 30-day views

Daily bars for week view, weekly buckets for month view.

Category breakdown

LinearProgressIndicators showing % of total spending per category.

Task completion gauge

Circular progress indicator showing productivity.

Export to PDF

Generate native PDF reports with the Android PdfDocument API.

Financial summary

Total In, Total Out, Net Balance, Average Daily Spending.

🔍 Global Search

Unified search bar

Searches across all modules simultaneously in real-time.

Filter chips

Filter results by All / Tasks / Notes / Money / Reminders / Ledgers.

Direct navigation

Tap a result card to jump straight to that specific item.

👤 Profile & Settings

Firebase Storage profile

Upload profile picture from camera or gallery.

Data export

Export all tasks, notes, reminders, and cashbook data to CSV.

Dark mode

Applies immediately and persists across launches safely.

Granular notifications

Independent toggles for Reminders, Digest, Tasks, Money, General.

Security

Change password, link Google auth, or completely delete account.
